# bankAPI
This is a API that can be used to make bank transactions.
1)Register new user
2)Check balance
3)Tranfer Funds
4)Take Loan
5)Repay Loan


Software Requiremnts
---------------------
1)Docker -- 
2)Docker Compose --
3)Postman -- To test the API


RUN on your local machine
--------------------------
sudo docker-compose build

sudo docker-compose up
